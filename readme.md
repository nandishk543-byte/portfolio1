# Nandish Kumar - Portfolio Website

## 📌 Project Description
This is a personal portfolio website built using Flask and MySQL.
It showcases certificates, skills, and includes a contact form that stores messages in a database.

---

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)
- MySQL
